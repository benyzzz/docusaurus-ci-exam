# Description détaillée du flux de travail Git Flow utilisé

j'ai utilisé Git Flow pour gérer les branches et les commits. Voici les étapes suivies :

1. Création d'une branche `develop` à partir de `main`.
2. Création de branches `feature` pour chaque nouvelle fonctionnalité.
3. Fusion des branches `feature` dans `develop` une fois terminées.
4. Fusion de `develop` dans `main` pour les releases.
